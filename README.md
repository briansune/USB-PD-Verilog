# USB-PD-Verilog

This example shows how to trigger USB Type-C Power Delivery via BMC "Biphase Mark Coding"

| Type	| Usage | 
| LUT	| 863	|
| FF	| 912	|
| IO	| 11	|
| BUFG	| 2	16	|
| MMCM	| 1	2	|
